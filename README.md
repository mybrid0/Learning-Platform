## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mybrid0/GitHub-Profiles-App
   cd Learning Platfrom

   ```

2. Install Dependencies:

   ```bash
   cd server
   npm install
   ```

   New Terminal

   ```bash
   cd client
   npm install

   ```

3. Start Server:

   ```bash
   cd server
   npm start
   ```

   New Terminal

   ```bash
   cd client
   npm start
   ```
